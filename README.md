# Showcase

This repository contains a small full-stack showcase project with a `backend` (Node/Express) and `frontend` (React).

Structure:
- `backend/` — server code
- `frontend/` — React app

To initialize and push:
1. `git init`
2. `git add .`
3. `git commit -m "Initial commit"`
4. `git remote add origin https://github.com/ksharma9848-jpg/Assignment-.git`
5. `git branch -M main`
6. `git push -u origin main`

If you see "fatal: not a git repository", run `git init` in the project root first.
